Detailed Description

A FastCGI request.

A request usually corresponds to a HTTP request that has been received by the webserver. You will probably want to use WriteResponse or its helper methods to output a response and then call Close. Use FCGIApplication.OnRequestReceived to be notified of new requests. Refer to the FastCGI specification for more details.
